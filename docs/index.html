快速开始

1

安装

把以下提示词发给你的 AI，让它帮你安全安装 HaluCatch

# 复制这段发给 AI
请先检查 ClawHub 上 skill 的元数据和安装要求。
如果这个 skill 要求安装第三方包或 CLI，请先验证来源、维护者和包内容，再执行安装命令。
只有在确认安全后，才从 ClawHub 安装 skill "HaluCatch"（@codermoray/halucatch）。
Skill 页面：https://clawhub.ai/codermoray/skills/halucatch
请把安装范围限定在这个 skill 内。
安装完成后，请根据 ClawHub 上验证过的元数据帮我完成设置。
只使用你在 ClawHub 上能验证的元数据，不要编造缺失的要求。
在做出任何更广泛的环境变更前，请先询问我。
2

审查你的 Skill

告诉 AI 你要审查哪个 Skill 目录

# 示例对话
用户：请用 HaluCatch 审查 /path/to/my-skill 目录

AI：📁 扫描目录...
    📝 发现 SKILL.md (120 行)
    🐍 发现 core.py (45 行)
    📊 生成三份报告...

AI：审查完成。报告已保存在 reports/ 目录：
    - 专业版：给工程师看的详细检查清单
    - 标准版：给业务人员看的通俗解释
    - 行动版：给 AI 的修复指令
3

阅读报告，决定下一步

三份报告分别面向不同角色，选择适合你的阅读

# 标准版 — 给业务/产品经理
用人话解释每个问题是什么、为什么重要、怎么改

# 专业版 — 给工程师/开发者
结构化检查清单，每个维度打分，精确到文件和行号

# 行动版 — 给 AI（直接复制粘贴）
修复清单 + 验证检查点，AI 按步骤执行即可

💡 如果决定修复：把行动版报告直接发给 AI，它会按清单逐项修复。修复后重新运行审查，确认问题清零。

1

克隆仓库

git clone https://github.com/CoderMoray/HaluCatch.git
cd HaluCatch
2

运行审查

# 自动检测语言（中文/英文）
python3 halucatch_core.py --skill-dir /path/to/your-skill

# 或强制英文输出
python3 halucatch_core.py --skill-dir /path/to/your-skill --lang en

# 或使用包方式
python3 -m halucatch --skill-dir /path/to/your-skill
3

查看报告

# 三份报告自动生成在 reports/ 目录
reports/HaluCatch-report-YYYY-MM-DD.md        # 专业版
reports/HaluCatch-report-YYYY-MM-DD-标准版.md  # 标准版
reports/HaluCatch-report-YYYY-MM-DD-行动版.md  # AI 行动版
